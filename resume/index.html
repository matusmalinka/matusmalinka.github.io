Name

matus — Senior Platform Engineer & Team Lead

Synopsis

Summary

Former Linux administrator, now a Platform engineer with a soft spot for open source. Focused on cloud-native application delivery built on containers and orchestration.

Currently Senior Platform Engineer / Team Lead — I build the pipelines, keep the whole infrastructure running, and automate every possible thing. I also bring new ideas, mentor the team, and own projects through to delivery.

Options

--admin

Install, administer and maintain Linux and UNIX systems — Red Hat, Debian, HP-UX, FreeBSD and Solaris derivations.

System and network configuration, monitoring and troubleshooting across SSH, FTP, DNS, DHCP, HTTPD and MySQL. Hands-on with high availability and NoSQL / key-value cache databases, mostly on the Elastic stack.

Advanced shell scripting; Python for the automation that deserves it. Comfortable across AWS, GCP and Azure.

--platform

Build and maintain CI/CD pipelines on GitHub, GitLab, Bitbucket and Jenkins. Manage, improve and monitor cloud infrastructure — mostly AWS, GCP and OpenStack.

Orchestrate Docker container clusters with Kubernetes, package with Helm, and run GitOps with ArgoCD. Monitoring and alerting with Prometheus and Grafana.

Infrastructure-as-code with Ansible, SaltStack and Terraform. CKA & CKS certified.
